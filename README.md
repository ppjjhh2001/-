# 안녕하세요, 끈기와 실력으로 신뢰를 만드는 개발자 박재현입니다! 🚀

### 📌 "2,190일의 성실함, 이제는 코드로 증명합니다."
저는 한 곳에서 **6년 동안 아르바이트**를 하며 대학 시절의 대부분을 보냈습니다. 단순히 시간을 채우는 것이 아니라, 매장의 단골 손님부터 재고 관리까지 제 손을 안 거치는 곳이 없을 정도로 깊이 몰입했습니다. 
어쩌다 한 번 결근하는 일 없이 묵묵히 자리를 지켰던 이 성실함은, 어려운 코드를 마주했을 때 도망가지 않고 끝까지 물고 늘어지는 저만의 **'뚝심'**이 되었습니다.

---

### 📂 주요 프로젝트: 지역소멸 대응 웹 서비스
> **"교수님도 포기하라던 RBAC, 집요함으로 끝내 구현했습니다."**

- **Project Summary:** 지역 소멸 위기 지역의 장소 소개 및 일자리 추천 서비스
- **Role:** 프론트엔드 개발 (사실상 1인 개발 및 프로젝트 리딩)
- **Key Experience:**
  - **위기 돌파:** 팀 내 공백으로 인해 혼자서 전체 구현을 도맡아 완수함.
  - **기술적 도전:** 기획 이후 설계 전체를 뒤엎어야 했던 **RBAC(역할 기반 권한 제어)**를 포기하지 않고 구현.
  - **사용자 중심:** 6년의 서비스 현장 경험을 녹여 사용자가 투덜대지 않는 직관적인 UI/UX 설계.

---

### 💡 저의 개발 철학

- **안정감 있는 개발자:** 제가 맡은 모듈은 팀원들이 신경 쓰지 않아도 될 정도로 완벽하게 관리합니다.
- **능동적인 협업:** '누군가 하겠지' 대신 **'내가 먼저 하겠다'**는 자세로 위기 상황을 돌파합니다.
- **현장 중심적 사고:** 서비스 현장에서 배운 '사용자 감각'을 코드에 녹여내어 진짜 편리한 화면을 만듭니다.
